# Relatório de Análise Exploratória de Dados (EDA)

**Base de dados:** `top_100_players.csv`  
**Contexto:** Análise do perfil e performance dos 100 jogadores mais bem ranqueados.  
**Objetivo:** Aplicar técnicas de EDA para descobrir padrões, correlações, outliers e segmentações visuais (clusters).  
**Entrega:** Relatório PDF + Notebook Python + Script `.py`

O notebook contém:
- Análise estrutural
- Limpeza de dados
- Feature engineering (quando aplicável)
- Histogramas, boxplots, correlação e clustering visual
- Insights iniciais e conclusões
